## 🆘 Troubleshooting

### Issue: Can't login
**Solution:** 
- Check Flask is running (Terminal 1)
- Check React is running (Terminal 2)
- Try email: `parent@example.com`

### Issue: Form won't submit
**Solution:**
- Check Flask terminal for errors
- Make sure all fields are filled
- Check browser console (F12)

### Issue: Can't find Behavior tab
**Solution:**
- Make sure you clicked on a child first
- Scroll right on tabs if on mobile

### Issue: AI not showing insights
**Solution:**
- Log at least 3 days of behavior
- Wait 2-3 seconds after logging
- Refresh page (F5)
